# Statistical-Analysis-of-Diabetes-Indicators

Descripción
-----------
Proyecto académico que realiza un análisis estadístico del riesgo de diabetes a partir de un conjunto de registros clínicos. El objetivo es identificar factores asociados, comparar grupos y explorar patrones en los datos mediante técnicas estadísticas y de aprendizaje no supervisado.

Qué se hace
-----------
- Análisis exploratorio de datos (estadísticas descriptivas y visualizaciones).  
- Pruebas de hipótesis para comparar grupos (t-test, ANOVA, chi-cuadrado u otras según el caso).  
- Modelado por regresión (regresión lineal para explicar variables clínicas como glucosa en ayunas).  
- Clustering con K‑Means para identificar subgrupos clínicos.  
- Reducción de dimensionalidad con PCA para visualización e interpretación.

Datos
-----
Datos obtenidos de Kaggle: "Diabetes Health Indicators Dataset" (conjunto amplio de variables demográficas, estilo de vida, antecedentes médicos y medidas clínicas). En el repositorio se presentan notebooks que muestran la carga, limpieza, análisis y modelos aplicados (no se incluyen datos privados).

Requisitos
---------
- Python 3.8+  
- Paquetes recomendados: pandas, numpy, scipy, scikit-learn, statsmodels, matplotlib, seaborn, jupyter

Instalación rápida
------------------
1. Crear entorno e instalar dependencias:
   ```
   pip install pandas numpy scipy scikit-learn statsmodels matplotlib seaborn jupyter
   ```
2. Abrir y ejecutar los notebooks en Jupyter o Colab.
